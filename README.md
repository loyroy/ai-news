## After pulling

1. git clone via SSH within Windows Linux enviroment.
2. cd into ai-news directory.
3. run: `docker run --rm --interactive --tty -v $(pwd):/app composer install`
4. run `sail npm install`
5. run `sail composer install`
6. copy .env.example to .env
7. run `sail artisan key:generate`
8. run `sail artisan migrate`
